💰 Tip Calculator

A simple and user-friendly Tip Calculator that helps split bills accurately among multiple people while factoring in tip percentages.

🚀 Features
Bill Calculation Based on Percentage
Easily calculate tips based on a selected percentage (e.g., 5%, 10%, 15%, etc.)
Split Between Multiple People
Divide the total bill (including tip) evenly among individuals
Dynamic Number of Persons
Increase or decrease the number of people sharing the bill
Real-Time Updates
Automatically recalculates totals as inputs change

🧮 HOW ITS WORKS
Enter the total bill amount
Select or input a tip percentage
Choose the number of people
The app calculates:
Total tip amount
Total bill including tip
Amount each person should pay
